# StyleStack

A full-stack outfit planning and recommendation web application built with React, TypeScript, and a REST API.

Goal: Build a scalable, production-style web application that supports closet management, outfit generation, planning, and personalization.

## Phase 1 – Closet + Manual Outfits
- Add/edit/delete clothing items
- Tag by type, color, season, occasion
- Filter/search closet
- Create and save manual outfits

## Phase 2 – Outfit Generator
- Generate outfit with constraints
- Basic scoring logic
- Exclude recent items

## Phase 3 – Planner + Personalization
- Calendar view
- Wear history
- Simple recommendation logic

## Future
- 3D outfit preview
- Experimental 3D-from-photo research
